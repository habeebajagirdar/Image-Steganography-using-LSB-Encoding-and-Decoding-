# Image Steganography using LSB Encoding and Decoding

## Description
A C application to hide and retrieve secret text inside a BMP image using the Least Significant Bit (LSB) technique.

## Technologies
- C
- File Handling
- Bitwise Operations
- Linux

## Compile
gcc *.c

## Run
-e beautiful.bmp secret.txt //encode
-d stego.bmp //decode
